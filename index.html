個人專案

成大選課小幫手

個人獨立開發 • May 2015 - Present

這是我學習 Rails 後第一份為解決實際問題而發起的專案， 學校選課系統和課程查詢系統的資訊整合不足、界面不佳，導致每次找課、排課都相當不便， 因為我沒有學校資料庫的存取權限，學校網站沒 API ，我當時也沒想到找學校談合作， 所以就先構想一個距離原本理想目標有段距離的折衷方案， 從官方網站現有缺陷切入，找尋只要我爬好課程資料，就可以改善的地方， 發現找課、排課功能有待改進。

網站五月初開始開發，六月底上線，根據 Google Analytics 的統計， 即時在線活躍使用人數最高曾到一百一十多人，並且在第一、第二階段選課期間內， 最高一天內的活躍使用人數曾到七百多人。

這個專案讓我很有成就感，收到許多同學和學弟妹的回饋， 也大大加強了自己走網站開發這條路的信心和興趣， 預計之後會加入分享修課經驗的功能，還有加強網站的 UI 。

MyFlix

Tealeaf Academy 課程作業 • December 2014 - March 2015

這是參加 Tealeaf Academy 第三個課程完成的專案， 是對於 NetFlix 的簡易 clone ， 練習內容主要是 TDD 、用 Rspec 撰寫測試、使用 AWS S3 完成檔案上傳功能、 串接 Stripe API 實踐刷卡付費功能、和運用幾種設計模式來簡化程式碼。
